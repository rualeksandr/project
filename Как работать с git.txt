1. В терминале открыть текушую папку cd
2. Говорим чтобы Git начал следить за папкой (иницируем git): git init
В папке с проектом создастся скрытая папка git
3. Прописываем пользователя который работает с git
Или локально (только для этого репозитория)
git config --local user.name "Alex"
git config --local user.email RuAleksandr07@gmail.com
Или глобально (для всех репозиториев на этом ПК)
git config --global user.name "Alex"
git config --global user.email RuAleksandr07@gmail.com
Чтобы убедиться что настройки применились в папке git в файле config они должны прописаться

Как работает git?

Три состояния файла:
1. Файл только создан, он не меняется
2. Git следит за определеными файлами (Они в индексе git)
3. Контрольная точка (комит) к ней можно вернутся

Узнать текущий статус файлов репозитория: git status

4. Добавляем в индекс git все файлы: git add -A
После добаления нового файла выполнить еще раз
5. Создаем коммит
git commit -a -m"Первая версия"

6. Смотрим какие коммиты были: git log

7. Подключаем внешний репозиторий
git remote add origin https://github.com/rualeksandr/project.git
git push -u origin master

Итог:
git add -A
git commit -a -m"Третья версия"
git push
